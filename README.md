# Image Gallery Project

A responsive Image Gallery website created using HTML, CSS, and JavaScript.

## Features
- Responsive image gallery layout
- Hover effects and smooth transitions
- Lightbox image preview
- Next and Previous navigation buttons
- Image category filters (Nature, City, Animals)
- Mobile-friendly design

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Description
This project is an interactive image gallery designed with a responsive layout. Users can view images in a lightbox, navigate using next and previous buttons, apply category filters, and enjoy smooth hover animations.

## How to Run
1. Download or clone the project
2. Open the `index.html` file in any browser
3. Explore the image gallery

## Project Objective
The objective of this project is to create a responsive image gallery with interactive features using front-end technologies.
